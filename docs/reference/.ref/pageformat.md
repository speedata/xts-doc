# Pageformat



The size of the PDF pages. Defaults to 210mm width and 297mm height (A4).



##  Child elements

(none)

##  Parent elements

[Layout](../layout.md)


## Attributes



`height` (length)
:   The height of the PDF pages.




`width` (length)
:   Width of the PDF pages.




## Example

```xml
<Pageformat width="150mm" height="150mm"/>
```

# DefineFontsize



Set the font size and the leading



##  Child elements

(none)

##  Parent elements

[Layout](../layout.md)


## Attributes



`fontsize` (length)
:   Fontsize, for example `12pt`.




`leading` (length)
:   Distance between two base lines, for example `14pt`.




`name` (text)
:   Set the symbolic name of the size.




## Example

```xml
<DefineFontsize name="title" fontsize="10pt" leading="12pt" />
```
